Zürich Transit Insights: Data Analysis and Geovisualization

Quick Links: App, Poster PDFs, GitHub Repo

  • This is a personal project motivated by a desire to expand my GIS toolset and a keen interest in the transportation system of Zürich.
    Also, this is an ongoing project, but take a look at how it's coming along!
    Steps:
    1. Data Analysis w/ Python & Pandas

    This put my programming and my German skills to the test! View the GitHub repo for this project here.

    2. Geospatial Analysis w/ ArcGIS Pro

    Including database queries (w/ SQL), spatial statistics, joins & relates, and other geoprocessing tools. Automation of tasks using ArcPy.

    3. Geovisualization w/ Adobe Illustrator

    Adding visual elements and creating an aesthetically pleasing design.

    4. Web Application

    I've recently created a web application to present the top 5 tram lines by ridership in an interactive, countdown format. View it here!

    5. Future plans...

    It's been really fun experimenting with different mediums for presenting the data I've aggregated thus far. But now I plan to go back to the data and do a deeper analysis!

  • Tools Used:
    • Python (inc. the Pandas and ArcPy libraries)
    • ArcGIS Pro
    • Adobe Illustrator
    • SQL
    • Microsoft Excel

    Skills Used:
    • Data Analysis
    • Geospatial Analysis
    • Database Queries
    • Joins & Relates
    • Spatial Statistics
    • Other geoprocessing tools such as Buffer & Dissolve operations

  • This project is sort of a part three to my overall exploration of Zürich, Switzerland. After participating in a UConn Experiential Global Learning course over the summer, my fascination with the city was ignited. Spending three weeks immersed in the Sustainable Cities course allowed me to delve into Zürich's renowned sustainability initiatives. My final project for the course, a short documentary entitled "Sustaining Minds, Sustaining Cities: Mental Health in Zürich" explores the impact of sustainability practices on the mental health of city residents. Watch it here!

    I continued my exploration of Zürich's sustainability by making it the focus of my final project for my Fall 2023 Intermediate German course. Leveraging my experience creating StoryMaps, I crafted a visual narrative highlighting Zürich's distinctive sustainability practices. This project can be viewed here.
